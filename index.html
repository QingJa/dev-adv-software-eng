🚀 自动化发布机器人运行指南

由于各大开放平台对个人自动发布有限制，我们为您编写了一个基于 Playwright 的本地自动化发布机器人。

# 1. 打开您的终端并安装 Playwright 依赖 pip install playwright playwright install # 2. 将刚下载的 publish_data.json 和 promo_video.webm 移入项目目录 # 3. 运行自动化脚本（xhs 小红书，dy 抖音） python publisher_bot.py -p xhs 或抖音版：python publisher_bot.py -p dy

工作原理：脚本会以有头模式自动加载 Chromium 浏览器并跳转到发布页面，引导您进行扫码登录。一旦检测到登录，机器人将自动上传生成的短视频，并自动填写生成好的食谱软文和标签，您只需最终确认发布即可！
