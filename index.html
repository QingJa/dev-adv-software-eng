🚀 个人平台发布助手运行指南

由于各大开放平台对个人自动发布有限制，这里提供一个基于 Playwright 的本地半自动发布助手，用于把用户计划内容填入个人小红书/抖音页面。

# 1. 打开您的终端并安装 Playwright 依赖 pip install playwright playwright install # 2. 将刚下载的 share_data.json 和 plan_share_video.webm 移入项目目录 # 3. 运行半自动分享脚本（xhs 小红书，dy 抖音） python publisher_bot.py -p xhs 或抖音版：python publisher_bot.py -p dy

工作原理：脚本会以有头模式自动加载 Chromium 浏览器并跳转到发布页面，引导您进行扫码登录。一旦检测到登录，助手将自动上传生成的计划打卡视频，并填写生成好的分享文案和标签；最终发布动作仍需用户自行确认。
